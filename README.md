# Portfolio Walkthrough
I have successfully created a modern, high-performance portfolio using **Astro** and **TailwindCSS**.
## Features Implemented
### 1. Modern Design System
- **Dark Mode**: A premium slate/zinc color palette with indigo/purple accents.
- **Glassmorphism**: Translucent backgrounds with blur effects on the navbar and cards.
- **Animations**: Smooth fade-ins, hover effects, and interactive elements.
### 2. Core Components
- **Header**: Sticky navigation with a mobile-responsive menu.
- **Hero**: Engaging introduction with a "Contact Me" call-to-action.
- **Skills**: Grid layout showcasing your tech stack (Astro, SvelteKit, etc.).
- **Projects**: Card-based layout to display your work with images and tags.
- **Contact**: A professional contact form layout.
### 3. Performance & SEO
- **Astro**: Zero-JS by default for the content, ensuring lightning-fast load times.
- **SEO**: Meta tags and semantic HTML structure.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
## How to Run
To start the development server and view your portfolio:
```bash
cd portfolio
npm run dev
